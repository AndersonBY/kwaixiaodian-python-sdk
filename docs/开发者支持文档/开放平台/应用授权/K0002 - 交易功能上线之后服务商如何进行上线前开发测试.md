# K0002 - 交易功能上线之后服务商如何进行上线前开发测试

具体场景：

该问题应对的具体场景为：三方工具类型-online上线状态-未上架到服务市场，即开放平台已上架，但服务市场未上架状态

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224117392.png)

#### 问题分类：

订阅授权类-code授权-获取授权码code

#### 问题现象：

应用上线前授权时提示“当前应用不允许此用户授权,原因:服务未订购，请进入服务市场订购后再进行授权操作 [100200101]”

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224150754.png)

#### 问题原因：

快手电商开放平台禁止用户线下交易，只有订购之后才能建立授权关系。如需测试可在服务详情中拿到链接进行订购授权测试，具体流程详见解决方案

#### 解决方案：

1、设置服务隐藏

服务市场中，新增/编辑服务时，服务属性-是否隐藏 ，选择 不公开服务

![](https://p2-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224185810.png)

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224221281.png)

2、设置套餐，提交审核

设置套餐时选择按时间收费，新增服务套餐，选择需要的有效期，套餐价格设置0.01（非试用版服务总价不能为0，此时请设置价格>0），点击提交审核

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224255242.png)

报错：“非试用版服务总价不能为0”，解决方法：请设置非7天套餐金额>0

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224289867.png)

3、请耐心等待运营同学审核

审核同学会在三个工作日内审核通过，请耐心等待

![](https://p2-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224326964.png)

4、审核通过，上架应用

审核通过后请点击“上架”按钮，上架需测试应用，此时应用已经上架，但并未展示在服务市场

![](https://p2-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224367304.png)

5、获取订购链接

请在服务列表中点击详情按钮，进入应用详情页面，点击服务访问地址，此时会跳转到订购授权页面

![](https://p2-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224401586.png)

如应用未审核通过会提示：“服务不存在或已下架”，请耐心等待审核

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224433606.png)

请复制该服务链接，样例：

![](https://p2-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224464088.png)

6、进行订购授权

此时请切换到商家登录，并选择需要的授权时限，并点击立即购买，进行订购授权，此时会返回code，请根据授权文档进行后续用授权码code换取长时令牌refreshToken以及访问令牌accessToken <https://open.kwaixiaodian.com/zone/new/docs/dev?pageSign=e1d9e229332f4f233a04b44833a5dfe71614263940720>

如未切换商家账号登录会报错：“只有店铺才可以购买服务 ”，请退出目前账号登录商家账号进行订购授权

![](https://p4-ec.ecukwai.com/kos/nlav10684/gravity-open-editor/daitianao/gravity-open-editor-1618224497669.png)
