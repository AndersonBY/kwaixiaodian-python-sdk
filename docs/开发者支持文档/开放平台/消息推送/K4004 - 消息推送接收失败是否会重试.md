# K4004 - 消息推送接收失败是否会重试

#### 问题分类：

消息推送类

#### 问题解答：

消息推送失败后，有两种重推机制

1.自动重推：快手电商开放平台发送消息给到设定的推送地址后，如果遇到失败(返回失败、网络异常等)，会自动重试5次，时间间隔为5, 10, 20, 30, 60分钟。

2.手动重推：开发者可以到“消息管理-消息查询”里手动操作需要重推的消息。
